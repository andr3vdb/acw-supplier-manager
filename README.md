# ACW Supplier Manager

A professional supplier management extension for AbanteCart 1.4.2.

## Project Status

🚧 Under Development

## Planned Features

- Product Export (CSV/XLSX)
- Supplier Import
- SMD Technologies Parser
- Rectron Parser
- Pinnacle Parser
- Mustek Parser
- Supplier Profiles
- SKU Matching
- Price Updates
- Stock Updates
- Import Preview
- Import History
- Rollback
- Scheduler
- Logging

## Requirements

- AbanteCart 1.4.2
- PHP 8.3+
- MariaDB
- Composer
